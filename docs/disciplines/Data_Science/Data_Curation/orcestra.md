# ORCESTRA

[ORCESTRA](https://orcestra.ca) is a platform that enables the creation of reproducible data objects for biomedical research. It integrates data from various sources, processes it using standardized pipelines, and packages it into versioned data objects that can be easily shared and cited.

You can read the official documentation and learn more about the platform on the [ORCESTRA Documentation Page](https://orcestra.ca/app/documentation/overview).
